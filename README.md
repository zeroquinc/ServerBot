# ServerBot

This is a Discord.py bot created for learning Python and for fun.
It is mainly used for Arrs and Plex notifications on your Discord server

# Purpose

- Plex Now Playing & New Content notifications
- Trakt Ratings notifications
- Trakt Weekly Global & User Top Shows / Movies notification
- Sonarr & Radarr Grab & Delete notifications
- Learning Python
- Enjoy building a Discord bot

# Usage

Make a .env and fill in these variables and save it in the root folder

```
# Discord Configuration
DISCORD_SERVER_ID =
DISCORD_TOKEN =

# Trakt API Configuration
TRAKT_CLIENT_ID =
TRAKT_CLIENT_SECRET =
TRAKT_USERNAME =

# The Movie Database (TMDb) API Key
TMDB_API_KEY =

# Tautulli Configuration
TAUTULLI_API_URL =
TAUTULLI_API_KEY =
TAUTULLI_USER_ID =
TAUTULLI_SCRIPTS_FOLDER =

# Discord Channel IDs
CHANNEL_PLEX_CONTENT =
CHANNEL_PLEX_PLAYING =
CHANNEL_RADARR_GRABS =
CHANNEL_SONARR_GRABS =
```
