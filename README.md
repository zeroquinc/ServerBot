## Usage

- Make a .env and fill in these variables

```
DISCORD_SERVER_ID=xxx
DISCORD_TOKEN = "xxx"
TRAKT_CLIENT_ID=xxx
TRAKT_CLIENT_SECRET=xxx
TMDB_API_KEY=xxx
TRAKT_USERNAME=xxx
TAUTULLI_API_URL = "http://xxx:8181/api/v2"
TAUTULLI_API_KEY=
TAUTULLI_USER=
```
